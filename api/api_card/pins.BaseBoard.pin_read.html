pins.BaseBoard.pin_read

pins.BaseBoard.pin_read¶

BaseBoard.pin_read(name, version: Optional[str] = None, hash: Optional[str] = None)¶

Return the data stored in a pin.

Parameters

  • name – Pin name.

  • version – A specific pin version to retrieve.

  • hash – A hash used to validate the retrieved pin data. If specified, it is compared against the pin_hash field retrived by pin_meta().
